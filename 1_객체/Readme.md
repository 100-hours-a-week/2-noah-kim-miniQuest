- **`[name]`**과 **`[age]`**는 해당 인스턴스의 속성값으로 대체되어야 합니다.
- 이름이 "Jane Doe"이고 나이가 25인 **`Person`** 인스턴스를 생성하세요.
- 생성한 인스턴스의 **`greet`** 메서드를 호출하여 메시지를 출력하세요.
