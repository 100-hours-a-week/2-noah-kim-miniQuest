### 1번 퀘스트

<aside>
👉 **`let`**과 **`const`**의 재할당 차이점 파악하기

1. **`let`**으로 선언된 변수 **`temperature`**에 처음에는 25를 할당하고, 이후에 30으로 값을 변경하세요.
2. **`const`**로 선언된 변수 **`MAX_TEMPERATURE`**에 35를 할당하고, 이후에 이 변수에 다른 값을 할당하려 시도했을 때 발생하는 오류를 설명하세요.
</aside>
